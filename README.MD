# Florería & Regalería "Petals & Co." - Proyecto Final

Este es el proyecto integrador final del módulo de Front End JS. Consiste en un e-commerce dinámico e interactivo diseñado para una florería y tienda de regalos, con una estética cuidada en tonos rosa claro, verde y blanco.

##Características del Proyecto
- **Estructura Semántica**: HTML5 accesible y optimizado para SEO con metaetiquetas clave.
- **Estilos y Responsividad**: Maquetación adaptativa mediante Bootstrap 5, CSS nativo, Flexbox (en tarjetas de productos), CSS Grid (en la sección de reseñas) y Media Queries.
- **API REST Integradora**: Consumo dinámico de productos mediante `fetch()`. Para adaptarlo a la temática de flores, se configuró una simulación controlada de datos con formato JSON profesional.
- **Carrito de Compras Completo**: CRUD del carrito en tiempo real (añadir, editar cantidades, eliminar de a un producto) persistido de forma local con `LocalStorage`.
- **Formulario de Contacto**: Validación de campos en el cliente con JavaScript y procesamiento de envíos mediante Formspree.
- **Multimedia y Accesibilidad**: Inclusión de mapas interactivos, textos alternativos (`alt`) en todas las imágenes y navegación fluida por teclado.
